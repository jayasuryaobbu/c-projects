# Electronic-Notice-Board

## Project Description:
  This project is designed to display short messages/notices which are received by the microcontroller from specified GSM Network.

## Components Used:
  * LCD 16x2 Display
  * Max232 IC
  * AT89S52 Micro Controller
  * 8051 Dev Board
  * Sim 800 GSM Module
